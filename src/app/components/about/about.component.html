Reglas del Solitario

Reglas

El juego que elegi es el solitario con las siguientes reglas:

Preparacion

Se usa baraja francesa
Se crean 7 pilas con distribución progresiva
Cartas restantes forman el mazo de descarte

Objetivo

Formar 4 pilas ordenados de mayor a menor por palo

Movimientos permitidos

Solo se mueve la última carta visible de cada pila
Pila vacía solo puede recibir un Rey
Se pueden mover grupos de cartas ordenadas como unidad
